## Команды для запуска

- сборка проекта - `npm run build`
- разворот проекта для разработки `npm run dev`
- запуск node.js для раздачи статики - `start-server`

## Стэк

- HTML
- Scss
- Handlebars
- Parcel
- Node v14.20.0
- Express

## Карта приложение

- Страница регистрации
- Страница авторизации
- Страница настроек пользователя
- Страница чатов
- Страница 404
- Страница 500
